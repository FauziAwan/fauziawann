Apa itu Socket?

Soket (socket) memungkinkan mesin untuk bertukar data dengan mesin yang lain.

Ketika komputer server dan komputer klien berkomunikasi, maka mereka harus membuat soket. Komunikasi tidak mungkin dapat dilakukan tanpa soket. Dalam jaringan komputer, soket dikenal sebagai ujung koneksi TCP. Soket ditentukan dengan alamat IP host dan nomor port. Pemrograman soket akan membantu anda memahami komunikasi antar mesin dalam jaringan.

Kita dapat melakukan pemrograman soket di linux menggunakan modul socket. Cobalah periksa, apakah modul ini tersedia di komputer anda. Masuk ke shell python kemudian impor modul socket.

Membuat Socket pada Python

Soket dapat dibuat dengan fungsi socket(). Fungsi ini terdapat dalam modul socket. Oleh karena itu, modul ini perlu kita import ke dalam skrip yang akan dibuat.

import socket

soket = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
print "soket telah dibuat"
