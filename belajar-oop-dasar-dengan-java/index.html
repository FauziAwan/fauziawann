Belajar OOP dasar dengan java

Ardianta Pargo

OOP (Object Oriented Programming) atau dalam bahasa indonesia dikenal dengan PBO (Pemrograman Berorientasi Objek) merupakan sebuah paradigma pemrograman yang menggunakan prinsip-prinsip objek.

Semua hal di dunia ini, dapat kita sebut sebagai objek. Objek memiliki ciri-ciri dan perilaku. Contohnya burung,memiliki ciri-ciri warna bulu kuning, memiliki perilaku bisa terbang, bisa berkicau dan lain-lain. Begitu juga dalam pemrograman, semua benda (objek) bisa kita gambarkan atau ciptakan dalam kode.

Pemrograman berorientasi objek tidak hanya dapat dilakukan dengan java. Tapi, dapat juga dilakukan dengna bahasa pemrograman lain seperti python, php, ruby, C++, dan bahasa-bahasa pemrograman lain yang mendukung OOP. Alasan memilih bahasa pemrograman java, karena java sendiri sudah menganut konsep PBO. Buktinya, setiap membuat program, kita harus membuat kelas terlebih dahulu. Mungkin di lain kesempatan, insa’allah akan saya bahas juga OOP dengan bahasa pemrograman yang lain.

Mengenal kelas

Kelas (Class) merupakan sebuah blue print (cetakan biru) dari sebuah objek. Dengan kata lain, kelas itu bisa kita ibaratkan sebagai kerangka dari objek. Kelas menyimpan informasi tentang ciri-ciri (atribut atau properti) dan perilaku (fungsi atau metode) dari objek. Kita bisa menciptakan objek baru dengan kelas. Contoh kelas:
Class Burung{
    // ciri-ciri
    string warnaBulu;
    string makanan;

   // tingkah laku
    void terbang(){
        System.out.println(“Terbang…”); 
    }

    void makan(){
        System.out.println(“Sedang makan “ + this.makanan); 
    }

}
