Koneksi PHP ke MySQL dengan PDO

Ardianta Pargo

PDO (PHP Data Object) merupakan salah satu ekstensi PHP untuk koneksi ke database. PDO tidak hanya mendukung DBMS MySQL, PDO juga mendukung DBMS lain seperti SQLite, PostgreSQL, Ms. SQL Server, ODBC, dsb. Pada kesempatan ini kita akan mencoba membuat kode koneksi PHP dengan database MySQL menggunakan PDO.

Membuat Koneksi ke MySQL dengan PDO

Koneksi PHP dengan MySQL menggunakan PDO membutuhkan DSN (Data Source Name).

<?php

$host = ‘localhot’;
$user = ‘root’;
$pass = ‘petanikode’;
$nama_db = ‘prtanian’;

$dsn = “mysql://host=$host;dbname=$nama_db”

$db = new PDO($dsn, $user, $pass);

?>

Maka kita telah membuat objek koneksi bernama $db. Selanjutnya kita tinggal manfaatkan objek tersebut untuk mengakses database. Objek $db akan menjadi perantara dalam mengakses database.
