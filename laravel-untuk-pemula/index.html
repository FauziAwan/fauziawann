Belajar Laravel: Pengenalan Dasar Laravel untuk Pemula

Salah satu Framework PHP yang populer di Indonesia adalah Laravel.

Bahkan Laravel menjadi repositori Github terpopuler di Indonesia:

![Repositori Github Terpopuler di asia]()

Laravel banyak digunakan untuk membuat aplikasi web, mulai dari aplikasi e-comerce, blog, forum, sistem informasi akademik, dan sebagainya.

Cara terbaik balajr Framework adalah dengan mencoba menerapkannya pada studi kasus nyata.

Pertama kita akan belajar cara meingstal Laravel dan berkenalan dengan struktur direktorinya.

Apa Saja yang Harus dipersiapkan untuk Belajar Laravel?

Ada beberapa hal yang harus dipersiapkan sebelum memulai belajar laravel, diantaranya:

Persiapkan Alat-alatnya:

  • Teks Editor (rekomendasi: VS Code).
  • Server PHP, Apache, dan MySQL (Baca: Persiapan Pemrograman Web di Linux)
  • Composer (Baca: Tutorial Cara Menggunakan Composer)
  • Nodejs, NPM, Webpack (untuk front-end)
  • Git (Opsional)

Persiapkan Pengetahuan dan Mental:

  • Sudah paham tentang pemrograman berorientasikan obejek (OOP) dengan PHP;
  • Sudah paham HTML, CSS (Bootstrap), dan Javascript.
  • Sudah paham cara menggunakan alat-alatnya.
  • Bisa Googling untuk mencari tahu;
  • Bisa Bahasa inggris;
  • Bisa Membaca dan Menulis Kode.

Instalasi Laravel di Linux dan Membuat Proyek Baru

Kita akan menginstal Laravel dengan Laravel Installer.

Membuat proyek baru Laravel

laravel new <nama proyek>

Contoh: laravel new tugas
