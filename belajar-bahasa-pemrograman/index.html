10 Pengetahuan Dasar untuk Menguasai Bahasa Pemrograman Apapun

Ardianta Pargo

Pada dasarnya setiap bahasa pemrograman itu sama. Menurut Wikipedia, Bahasa Pemrograman adalah bahasa formal yang didesain untuk memberikan instruksi ke mesin (komputer). Bahasa pemrograman dapat digunakan untuk membuat program yang mengendalikan tingkah laku mesin atau untuk menerapkan sebuah algoritma. Nah, pada kesempatan ini, Saya akan berbagi pengetahuan dasar untuk menguasai bahasa pemrograman.

1. Persiapan Sebelum Belajar Pemrograman

Hal pertama yang perlu diketahui sebelum memulai belajar bahasa pemrograman adalah cara mempersiapkan peralatan yang dibutuhkan. Apa saja peralatan yang dibutuhkan dan bagaimana cara menggunakannya? 2. Aturan Sintak

Setelah mempersiapakan alat-alat untuk belajar, selanjutnya kita perlu mengetahui aturan-aturan penulisan sintak. Setiap bahasa pemrograman memiliki aturan yang berbeda-beda. Berikut ini beberapa aspek yang perlu diperhatikan dalam aturan penulisan sintak.

a. Struktur Baku

Struktur baku merupakan format penulisan yang harus diikuti. Misal, pada bahasa pemrograman Java, Kita diharuskan membuat kelas terlebih dahulu. Pada pemrograman pascal kita harus mendeklarasikan variabel-variabelnya terlebih dahulu. Kemudian pada PHP, harus ada <?php di setiap awal kode.

Beberapa post terkait:

  • Struktur Kode Bahasa Pemrograman Pascal
  • Struktur Kode Bahasa Pemrograman C
  • Struktur Kode Bahasa Pemrograman Python
  • Struktur Kode CSS (Berkenalan dengan CSS)
  • 3 Cara Penulisan Kode CSS

b. Kata-kata Terlarang

Kata-kata relarang (reserved word) yang dimaksud adalah kata-kata yang tidak boleh digunakan sebagai nama variabel maupun objek. Misal, kata-kata terlarang dalam bahasa C++ adalah: if, elif, else, endif, ifdef, ifndef, define, selengkapnya lihat di: http://en.cppreference.com/w/cpp/keyword

c. Penulisan Komentar

Komentar merupakan bagian yang tidak akan dieksekusi. Kita bebas memberikan komentar apa saja dalam kode. Namun, perlu diperhatikan peraturan penulisannya. Misal dalam bahasa pemrograman java menggunakan garis miring dua kali (//) di setiap awal komentarnya. Sedangkan di python menggunakan tanda pagar (#). Setiap bahasa pemrograman memiliki aturan yang berbeda-beda.

Java, PHP, Javascript, C, C++:

// ini adalah sebuah komentar

Python, Ruby, PHP

# Ini adalah sebuah komentar

HTML

<!-- Ini adalah komentar -->

CSS

/* Ini adalah komentar */
