Pemrograman Javascript: Langkah Awal Belajar Javascript

#Pemrograman Web #Javascript

Ingin belajar javaScript? tapi tidak tahu memulai dari mana. Tulisan ini sangat tepat untuk anda. Tulisan ini akan memandu anda untuk belajar javascript dari awal. Saya harap, anda sudah belajar atau mengetahui HTML dan CSS sebelumnya, karena akan sangat membantu dalam memahami javascript. Sebelum memulai belajar sebaiknya berdoalah terlebih dahulu, semoga diberikan ilmu yang bermanfaat.

Apa itu javascript?

Javascript adalah bahasa pemrograman yang digunakan pada web browser. Javascript termasuk dalam scripting language, artinya javascript tidak perlu dikompilasi untuk menjalankannya. Javascript semakin berkembang, sekarang javascript tidak hanya digunakan pada browser. Javascript juga dapat digunakan untuk pemrograman Desktop (GUI), Mobile,Web Backend (seperti php), dan lain-lain.

Alat yang diperlukan untuk belajar javascript

Kita membutuhkan Teks Editor untuk menulis kode JavaScript dan web browser untuk melihat hasil output. Dalam tulisan ini, saya menggunakan Teks Editor: Mousepad dan web browser: Google Chrome dan Mozilla Firefox yang berjalan di atas Xubuntu.

Console JavaScript

Ada yang mengatakan, belajar javascript itu susah. karena saat melihat hasil di web browser, pesan error-nya tidak tampil. Pendapat tersebut tidak benar, karena berkat Console Javascript, kita dapat melihat pesan error yang terjadi. Console javascript dapat di akses melalui web browser. Pada google-chrome, klik kanan lalu pilih inspect element atau dengan menekan tombol F12. Setelah itu masuklah ke console.
Cara mengakses Console di Mozilla Firefox juga sama seperti di Google Chome.
Kita dapat menulis perintah atau kode-kode javascript di dalam console. Hasilnya pun langsung di tampilkan. Console terkadang dipakai untuk debugging (mencari error) dan uji coba kode.

Penulisan javascript di HTML

Kode Javascript di HTML dapat di tulis di dalam tag <script>. Jangan lupa juga untuk menambahkan atribut type=‘text/javascript’ untuk menandakan kode yang di dalam tag tersebut adalah kode javascript. Contoh:
<script type=‘text/javascript’>
    document.write(‘Assalamulaikum, ini dari JavaScript’);
    document.write(’<br/>’);
    document.write(‘Saya sedang belajar javascript.’);
</script>
Hasilnya:
Tag <script> dapat di tulis di dalam tag <head> atau <body>. Pada contoh di atas, terdapat kode atau perintah document.write(), kode ini berfungsi untuk menulis sesuatu di dokumen (html). Ada juga perintah console.log(), perintah ini untuk menulis seuatu pada Console, dan ada lagi window.alert() atau alert() saja. Perintah ini untuk menulis pesan pada jendela alert. Contoh:
<!DOCTYPE html>
<html>
<head>
    <title>Belajar Js</title>
    <script type=‘text/javascript’>
         alert(‘Assalamualaikum’);
         window.alert(‘Saya sedang belajar javascript’);
    </script>
</head>

<body>

<script type=‘text/javascript’>
     document.write(‘tulisan ini akan tampil di dokumen HTML’);
     console.log(‘tulisan ini akan tampil di dalam Console’);
</script>

</body>
</html>
