Seperti Apa Pemrograman PHP itu?

PHP merupakan bahasa skrip (scripting langage) yang berjalan di atas server. Server dapat kita buat di komputer kita sendiri dengan bantuan Apache maupun XAMPP. Skrip PHP akan diskekusi di server setiap ada permintaan HTTP (HTTP Request) dari Client (Web Browser), kemudian membalas permintaan HTTP tersebut dengan dokumen HTML. Sehingga web bisa ditampilkan pada Browser.

Program Halo Dunia!

Mari kita membuat sebuah program Halo Dunia! (Hello World!) agar lebih paham cara program PHP bekerja. Buatlah sebuah file baru dengan teks editor bernama halo_dunia.php, kemudian isi dengan kode beikut.
<?php echo “Halo Dunia!”; ?>
Setelah itu, simpan dalam direktori server pada /var/www/html. Kalau pakai XAMPP di windows, simpan di C:\xampp\htdocs. Kemudian cara membuka programmnya adalah dengan mengakses alamat http://localhost/helo_dunia.php melalui browser.

[web sedang mengeksekusi php]

Apabila berhasil tampil seperti pada gambar di atas, artinya program kita sudah berhasil berjalan di server. Program di atas adalah program yang paling sederhana. Hanya terdiri dari satu intruksi atau perintah. Selanjutnya coba tambahkan dua baris perintah lagi.

<?php
echo “Halo Dunia!”;
echo “Nama saya, Petani Kode”;
echo “Saya sedang belajar PHP”;
?>

Simpan! Kemudian coba eksekusi atau jalankan lagi. Hasilnya akan seperti apa? Coba lihat saja tangkapan layar dibawah ini.
[Screenshoot sedang menjalankan PHP]
Ternyata, teksnya ditampilakn dalam satu baris. Padahal dalam kode, kita membuat tiga baris. PHP tidak peduli dengan hal itu. Untuk memberi tahu PHP, agar teks ditampilakn menjadi tiga baris. Kita harus menambahkan \n di setiap akhir dari teks yang akan dicetak/ditampilkan. Sehingga kodenya menjadi seperti berikut ini.
echo “Halo Dunia!\n”;
echo “Nama saya, Petani Kode\n”;
echo “Saya sedang belajar PHP\n”;
Tanda \n berfungsi membuat baris baru. Dengan demikian, PHP mencetaknya menjadi seperti berikut ini.
