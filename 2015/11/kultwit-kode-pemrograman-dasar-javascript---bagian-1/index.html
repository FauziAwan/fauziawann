KulTwit Kode: Pemrograman dasar Javascript - Bagian 1

Ardianta Pargo dalam pemrograman
22 November 2015 · 05:30

Kultwit berarti Kuliah Lewat Twitter atau twitter bersambung tentang suatu topik. Seorang penulis bernama Gunawan Mohammad yang pertama kali mencetuskan istilah ini melalui akun twitternya  @gm_gm.

Malam ahad, 21 Nopember 2015. Akun twitter Petani Kode (@PetaniKode) mengadakan kultwit kode. Kultwit ini pertama kali dilakukan. Insa’allah ke depan akan terus dilakukan. Topik yang dibahas pada kultwit adalah pengenalan dasar pemrograman javascript. Berikut ini adalah rangkuman twitnya.

Siap-siap #ngodejs, kultwit akan segera dimulai!
— Petani Kode (@petanikode) November 21, 2015

Agar twit lebih mudah dikumpulkan, maka digunakan hastag #ngodejs. Selamat menikmati kultwit!

#ngodejs (1) Alat-alat yang kalian butuhkan untuk pemrograman #javascript: (I) Web browser, (II) Teks Editor.
— Petani Kode (@petanikode) November 21, 2015

Pada kultwit ini, web browser yang digunakan adalah Mozilla Firefox dan teks editornya adalah Mousepad. Anda juga bisa menggunakan web browser lain seperti Google Chrome dan teks editor serupa seperti Notepad++, Gedit, Notepad dll.

#ngodejs (2) Web browser adalah tempat #javascript berjalan. Kita juga bisa mengakses #console javascript di sana.
— Petani Kode (@petanikode) November 21, 2015

Konsol javascript merupakan sebuah antar muka berbasis teks yang memungkinkan kita untuk memberikan fungsi-fungsi javascript secara langsung di sana.

#ngodejs (3) Teks editor adalah alat yang kita pakai menulis #kode #javascript.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (4) Bila tidak ingin repot-repot mempersiapkan dua alat tersebut. Teman-teman bisa memanfaatkan https://t.co/ACK6U4f3kD
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (5) Pengetahuan HTML, CSS, dan pengalam ngode di bahasa pemrograman lain akan membantu anda cepat belajar pemrograman #javascript
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (6) Baiklah, mari kita mulai belajar #javascript-nya!
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (7) Jadi, apakah itu #javascript ?
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (8) #Javascript adalah bahasa pemrograman yang diciptakan oleh @BrendanEich. Awalnya berjalan di atas browser @Netscape
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (9) #javascript pernah dijuluki Mocha, LiveScript ref: https://t.co/2sLTyzusw4
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (10) #javascript menggunakan standar @ECMAScript
— Petani Kode (@petanikode) November 21, 2015

ECMA Script adalah standar untuk bahasa pemrograman skrip (Scripting language) seperti Javascript, Action Script, dll. Sekarang standarnya sudah sampai versi 6.

#ngodejs (11) Baik, perkenalan #javascript cukup sampai di situ saja. Sekarang mari kita mulai #ngodejs, buka web browser masing-masing!
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (12) Saya akan perkenalakan dengan #konsol #javascript. Klik kanan di browser, kemudian pilih “Inspect Element”
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (13) Setelah itu, masuk ke konsol dengan mengelik “Console” pic.twitter.com/qxNUCsuUCq
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (14) Coba ketik: console.log(“Halo, saya sedang belajar javascript!”) pic.twitter.com/jaqPVmCSia
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (15) Konsol #javascript langsung merespon kode yang kita tulis. Konsol akan mengerti kode yang kita inputkan.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (16) Bila salah input kode, maka konsol akan memberitahu. Contoh: input “petani”, output “ReferenceError: petani is not defined”
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (17) Itu karena kata kuci (kode) “petani” tidak dikenali di #javascript.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (18) Sekarang coba ketik: alert(“Sedang ikut kultwit #javascript di @petanikode”) pic.twitter.com/AOT7cv2GKR
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (19) Bagaimana, apa sudah paham cara menggunakan konsol #javascript sekarang?
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (20) Konsol #javascript sering digunakan untuk mengetes kode dan debugging (pencarian error).
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (21) Konsol #javascript juga mampu menampilkan error dari kode js yang kita tulis dan menunjuukan posisi errornya.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (22) Selanjutnya kita akan mencoba menulis kode #javascript di #html
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (23) Menulis kode #javascript di #html ada tiga cara: 1) Internal, 2) Inline, dan 3) Eksternal
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (24) Kode #javascript internal, artinya kode JS ditulis di dalam tag <script>
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (25) tag <script> bisa ditaruh di dalam tag <head> dan tag <body>
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (26) Banyak yang menulis tag <script> di akhir dokumen atau sebelum penutup tag <body>. Alasannya, biar website lebih cepat di load
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (27) Kemudian penulisan #javascript inline, artinya JS ditulis di dalam atribut tag. Contoh: <b onclick=“alert(‘Js’)”>Klik!</b>
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (28) Kode #javascript inline biasanya digunakan untuk memanggil fungsi pada event tertentu. Misal: onclick, onload, onsubmit, dll.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (29) Lalu, eksternal js adalah kode #javascript yang ditulis terpisah dengan file #html
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (30) File eksternal #javascript disimpan dengan ekstensi “.js”. Contoh: petani-kode.js
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (31) Untuk me-link-kan (menyambung) file #html dan #javascript, kita hanya perlu manambahkan atribut “src” pada tag <script>.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (32) Contoh: <script src=“petani-kode.js”></script> Degan demikian, file #HTML dan #javascript sudah di link
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (33) Nah, itulah tiga cara menulis kode #javascript di #HTML. Manakah yang kalian suka?
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (34) Sekarang kita akan coba menulis kode #javascript internal dan inline. Silahkan buka teks editor atau https://t.co/ACK6U4f3kD
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (35) Buatlah kodenya seperti ini https://t.co/N5YL3dfHMS pic.twitter.com/JHqnTno81D
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (36) Ketika dibuka di browser web, maka akan mendapatkan hasil seperti ini pic.twitter.com/C2seFWpXlz
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (37) Ketika teks “Javascript” diklik (event klik) maka akan keluar jendela dialog menampilkan pesan “Ouch!! JS!”.
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (38) Perhatikan juga di konsol #javascript, di sana akan tampil teks berdasarkan yang dituliskan di fungsi: console.log()
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (39) Kita baru saja mengetahui dua fungsi di #javascript: alert() dan console.log()
— Petani Kode (@petanikode) November 21, 2015

#ngodejs (40) #kutwitkode malam ini kita akhiri sampai di sini. Insa’allah akan dilanjutkan di waktu mendatang. Tetap pantau @petanikode!
— Petani Kode (@petanikode) November 21, 2015
