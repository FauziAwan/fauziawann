KulTwit Kode: Pemrograman dasar Javascript - Bagian 2

Ardianta Pargo dalam pemrograman
22 November 2015 · 17:17


Melanjutkan #kultwitkode sebelumnya dengan topik yang sama. Namun, pembahasannya berlanjut. Pada kultwit kedua ini, yang dibahas adalah javascript eksternal, tata bahasa javascript, DOM (Document Object Model), dan sedikit tentang variabel dan tipe data. Berikut ini adalah rangkaian twitnya, selamat menikmati!

Siap-siap, kita akan segera melanjutkan #kultwitkode tadi malam. Masih dengan topik yang sama #pemrograman dasar #javascript, ayo #ngodejs
— Petani Kode (@petanikode) November 22, 2015

Tadi malam kita sudah #ngodejs, mencoba internal dan inline #javascript. Sekarang kita akan coba menggunakan eksternal javascript.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Sekarang coba buat file baru berdampingan dengan file #HTML-nya. Berikan nama petani-kode.js pic.twitter.com/xh4GeUawcH
— Petani Kode (@petanikode) November 22, 2015

#ngodejs file petani-kode.js akan berisi kode #javascript. Kemudian, nanti kita link-kan dengan file #HTML-nya
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Silahkan tambahkan tiga baris kode #javascript (di gambar) ini pada file petani-kode.js pic.twitter.com/UnYwCYDSHD
— Petani Kode (@petanikode) November 22, 2015

#ngodejs pada file kultwitjs.html tambahkan kode berikut untuk me-link-kan: <script src=“petani-kode.js”></script> pic.twitter.com/mLdS7gbffb
— Petani Kode (@petanikode) November 22, 2015

menyambungkan file javascript yang tersimpan di satu tempat dengan file HTML dapat langsung dengan menuliskan nama file javascript-nya. Tapi, bila terletak di direktori yang berbeda maka kita juga perlu menyebut nama direktori itu.

#ngodejs Coba buka dengan web browser, lihat dan perhatikan apa yang terjadi. pic.twitter.com/8SP9ega4M9
— Petani Kode (@petanikode) November 22, 2015

#ngodejs window.alert() merupakan fungsi untuk menampilkan pesan melalui jendela dialog
— Petani Kode (@petanikode) November 22, 2015

#ngodejs document.write() merupakan fungsi untuk menulis teks ke dokumen (#html). Fungsi ini disediakan oleh #DOM (Document Object Model)
— Petani Kode (@petanikode) November 22, 2015

#ngodejs #DOM merupakan sebuah #API yang menyediakan fungsi-fungsi #javascript untuk memanipulasi dokumen seperti #html, #xml, dan #svg
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Kemudian, console.log() berfungsi untuk menulis teks ke konsol. Biasanya digunakan untuk debugging dan testing.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Selanjutnya, kita akan belajar tata #bahasa #javascript. Seperti apa sih aturan-aturan penulisan kode javascript?
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Setiap baris kode (perintah) di #javascript harus diakhiri dengan titik koma (;)
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Krakter di #javascript bersifat sensitif. Artinya, kata “Petani” dan “petani” itu berbeda.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Selanjutnya pembuatan komentar, komentar merupakan bagian yang tidak akan dieksekusi oleh komputer.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Komentar sangat membantu untuk memahami maksud kode.
— Petani Kode (@petanikode) November 22, 2015

Komentar memang akan diabaikan oleh komputer. Tapi sangat membantu untuk memahami maksud kode. Bayangkan, bila anda bekerja di tim. Anda tidak menulis komentar sama sekali. Apakah orang lain (anggota tim) akan mampu memahami kode yang anda tulis? Oleh karena itu, kita perlu membuat komentar.

#ngodejs Komentar di #javascript ada dua jenis: 1). komentar satu baris, dan 2). Komentar yang lebih dari satu baris
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Komentar satu baris dapat dibuat dengan garis miring ganda (//), contoh: // ini adalah komentar
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Komentar yang lebih dari satu baris dapat dibuat dengan garis miring dan bintang. Contoh: /* ini komentar yg lebih dari 1 brs /
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Sekarang buka kembali file petani-kode.js, tambahkan sebuah komentar seperti pada gambar. pic.twitter.com/vXCaSNMbSd
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Perhatikan!, komentar tidak diakhiri dengan titik koma (;). Karena komentar bukan perintah.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Komentar yang lebih dari satu baris di mulai dengan tanda / kemudian diakhiri dengan tanda */
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Selanjutnya, saya akan memperknalkan anda dengan variabel dan tipe data #javascript
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Apa itu variabel dan tipe data?
— Petani Kode (@petanikode) November 22, 2015

#ngodejs variabel adalah sebuah nama tempat menyimpan sebuah data. Sementara itu, tipe data adalah jenis data yg tersimpan dalam variabel.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Pembuatan variabel di #javascript diawali dengan kata kunci “var” atau boleh juga tidak.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Contoh: var namaSaya = “Petani Kode”; usia = 20; Maka komputer akan mengingat variabel di atas serta nilainya.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs simbol sama dengan (=) digunakan untuk mengisi nilai variabel. Untuk nilai yg berupa teks diapit dengan tanda petik.
— Petani Kode (@petanikode) November 22, 2015

Simbol sama dengan (=) disebut juga operator pengisian. Operator yang fungsinya untuk mengisi nilai ke variabel.

#ngodejs Sementara itu, untuk nilai variabel yang berupa angka tidak diapit dengan tanda petik.
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Sekarng, cobalah membuat variabel di file petani-kode.js seperti pada gambar pic.twitter.com/obDiVBmDOu
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Perhatikan apa yang terjadi, isi variabelnya ditampilkan ke web. pic.twitter.com/9oHLmq4S6e
— Petani Kode (@petanikode) November 22, 2015

#ngodejs Nah, itu sedikit pengenalan variabel. #kultwitkode sore ini kita akhiri sampai di sini. Insa’allah akan dilanjutkan lagi. :)
— Petani Kode (@petanikode) November 22, 2015


Sekain untuk kultwit kedua ini, semoga bermanfaat. Nantikan kultwit selanjutnya, pantau terus timeline @PetaniKode!
