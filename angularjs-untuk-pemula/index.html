Belajar AngularJS: Pengenalan Angularjs untuk Pemula

Ardianta Pargo

Apa itu AngularJS?

AngularJS merupakan sebuah framework javascript untuk membangun UI atau antarmuka web.

Aplikasi Hello World

Pada contoh ini, kita akan membuat sebuah aplikasi sederhana. Mari membuat sebuah direktori baru bernama hello-angular. Isilah direktori tersebut dengan berkas angular.js yang telah diunduh. Kemudian buatlah berkas HTML baru bernama hello.html, isinya sebagai berikut:

<!DOCTYPE html>
<html>
<head>
<title>Hallo Angular</title>
<script src="angular.js"></script>
</head>

<body>
<div ng-app="helloAngular">
<p>Nama: <input type="text" ng-model="nama"></p>
<h1>Hallo, {{ nama }}, Selamat datang di AngularJS</h1>
</div>

<span class="Apple-tab-span" style="white-space: pre;">
</span>


</body>
</html>

Kode <script src="angular.js"></script> Berfungsi meload pustaka Angular ke dalam HTML. Kita juga dapat menggunakan CDN Google, yaitu: <http://ajax.googleapis.com/ajax/libs/angularjs/1.4.3/angular.min.js>.

Atribut ng-app pada elemen div menandakan elemen tersebut milik angular. Elemen tersebut akan dikelola oleh Angular. Atribut ini berisi nama aplikasi. Pada contoh, saya menggunakan helloAngular.

Atribut ng-model berfungsi untuk membuat model. Model bisa kita katakan sebagai variabel yang menyimpan data sementara. Pada contoh diatas, saya membuat model bernama nama pada elemen input.

Tanda kuring kurawal ganda {{ ... }}, berfungsi untuk membinding data dari model.
