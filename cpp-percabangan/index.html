Sturktur Pemilihan atau sering juga disebut percabangan membuat komputer dapat berpikir dan memilih suatu kondisi tertentu. Ada dua struktur pemilhan dalam prmrograman C++, IF-ELSE dan SWITCH-CASE. Untuk lebih jelasnya langsung saja mencoba contoh program di bawah ini.

Program Transakasi Pembayaran

Dalam program ini, kita akan membantu kasir menghitung total pembayaran dari pembeli yang mendapatkan diskon. Ketentuan yang berlaku di toko adalah “jika belanja lebih dari Rp 100.000 maka dapat diskon 20%”. Program ini implementasi dari struktur pemilihan IF-ELSE, Dibawah ini adalah kode programnya.
#include <iostream>
using namespace std;

int main(){
    cout << “Program Pembayaran” << endl;
    unsigned int totalBayar;
    float diskon = 20;
    unsigned int totalBelanja;

   cout << “Masukkan Total Belanjaan: “;
   cin >> totalBelanja;
   
   if(totalBelanja > 100000){
        totalBayar = totalBelanja - (totalBelanja * diskon/100);
   } else {
        totalBayar = totalBelanja;
   }

   cout << “Total Bayar: Rp “ << totalBayar << endl;
   return 0;
}
