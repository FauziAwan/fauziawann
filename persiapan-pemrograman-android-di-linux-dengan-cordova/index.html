Persiapan Pemrograman Android di Linux dengan Cordova

Ardianta Pargo

Cordova merupakan sebuah framework yang memungkinkan kita membuat aplikasi mobile dengan bahasa web seperti HTML 5, CSS 3, dan Javascript. Dengan menggunakan Cordova, kita dapat membangun aplikasi mobile tidak hanya untuk satu flatform.

Cordova (dulunya adalah PhoneGap), awalnya dibuat oleh perusahaan bernama Nitobi. Pada tahun 2011 kemudian Adobe System membeli perusahaan tersebut, dan menyerahkan pengembangan open-source-nya kepada Apache Software Foundation dengan nama baru Apache Cordova. [Tony Harianto, Codepolitan]

Pada kesempatan ini, saya akan membahas persiapan yang dilakukan untuk pemrograman aplikasi mobile dengan Cordova. Silahkan di simak!

1. Instal NodeJs dan NPM 2. Instal Cordova 3. Instal JDK 4. Instal Ant 5. Instal Android SDK 6. Update SDK 7. Membuat Proyek 8. Run di Browser 9. Membuat Paket APK
