Tutorial Cara Membuat Fitur Upload File/Foto di CodeIgniter

Ardianta Pargo

Upload file/foto merupakan fitur yang harus anda kuasai cara pembuatannya dalam pemrograman web. Karena dengan fitur ini, pengunjung atau user dapat mengirim file/foto ke server. Pada kesempatan ini, saya akan membahas cara membuat fitur upload di CodeIgniter. Selamat menyimak!

1. Persiapkan Tabel di Database

Pastikan anda sudah menyiapkan tabel di database. Tabel yang saya persiapkan dalam tutorial ini, bentuknya seperti gambar berikut ini.

Kode SQL:
CREATE TABLE barang (
  id_barang varchar(16) NOT NULL,
  nama varchar(64) NOT NULL,
  merek varchar(64) NOT NULL,
  kategori varchar(32) NOT NULL,
  keterangan text NOT NULL,
  harga varchar(16) NOT NULL,
  foto varchar(32) NOT NULL
) ENGINE=InnoDB DEFAULT CHARSET=latin1;

  ADD PRIMARY KEY (id_barang);
