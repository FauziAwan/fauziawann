Instalasi dan uji coba Lazarus IDE di Windows 7

#Lazarus #Windows #Pascal

Lazarus merupakan sebuah IDE open source yang mampu berjalan di berbagai sistem operasi (platform). Pada tulisan ini, saya akan memandu anda menginstal lazarus di Windows 7. Lazarus yang akan diinstal ialah Lazarus versi 1.4.4.

1. Download Lazarus

Silahkan menuju ke website resminya Lazarus untuk mendapatkan instalator Lazarus untuk windows.

Masuklah ke halaman download, kemudian pilih link Windows (32 and 64 bits) Direct download.
Proses download pun dimulai, silahkan menunggu dengan sabar!

2. Exsekusi Instalator

Klik ganda pada instalator lazarus (lazarus-1.4.4.-fpc-2.6.4-win.exe) untuk memulai instalasi. Pada jendela yang muncul, klik saja tombol Run.
Kemudian jendela pemilihan bahasa akan muncul, pilih saja English. Kemudian klik OK.

Kemudian akan muncul jendela sambutan selamat datang, klik Next untuk melanjutkan.

3.Lokasi instal

Menentukan lokasi tempat Lazarus akan terinstal, biarkan saja di C:\lazarus, kemudian klik Next.

4. Memilih komponen

Pilih Full instalation agar semua komponen terinstal. Setelah itu, klik Next.

5. Menentukan direktori menu

Direktori menu merupakan nama direktori pada Start Menu. Biarkan saja apa adanya, kemudian klik  Next.

6. Pengaturan tambahan

Silahkan centang Create desktop icon agar ikon Lazarus ditampilkan di desktop. Setelah itu, klik Next.

7. Informasi rangkuman

Jendela ini berisi informasi rangkuman instalasi yang akan dilakukan. Klik saja Install untuk memulai instalasi.

8. Proses Instalasi

Proses instalasi pun dimulai, silahkan menunggu dengan sabar sampai instalasinya selesai. Proses ini tidak membutuhkan waktu yang lama.

9. Instalasi Selesai

Klik Finish untuk mengakhiri instalasi.


—

Instalasi telah selesai di lakukan. Selanjutnya, silahkan coba buka Lazarus yang sudah terinstal. Karena baru pertama dibuka, maka anda akan mendapatkan jendela ucapan selamat datang di Lazarus. Tidak perlu merubah apa-apa, langsung saja klik Start IDE.

Selamat, lazarus sudah siap digunakan untuk membuat program

Uji Coba Lazarus

Pada uji coba ini, saya membuat program sederhana. Program ini hanya untuk menampilkan teks “Petani Kode” saat tombol Proses diklik.
