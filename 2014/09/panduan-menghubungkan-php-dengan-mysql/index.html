Panduan menghubungkan PHP dengan MySQL

Ardianta Pargo dalam pemrograman
10 September 2014 Â· 06:30


Panduan ini dibuat di linux. Bagi pengguna windows, silahkan menyesuaikan.

Membuat basis data baru

Gunakan phpmyadmin untuk membuat basis data baru. Buka phpmyadmin (http://localhost/phpmyadmin), buatlah sebuah basis data baru dengan nama petani.

Setelah itu, membuat tabel anggota dengan kolom (field) id, nama, dan alamat.

Kemudian isi kolom-kolom field sebagai berikut, setelah itu klik tombol save atau simpan.

Setelah itu, masukkan sebuah data. Klik sisipkan atau insert.

Kode PHP untuk menghubungi MySQL

Buatlah sebuah berkas php baru, lalu isi dengan kode program berikut ini.

Simpan kode tersebut di server web dengan nama konek.php. Simpan pada direktori /var/www

Penjelasan kode

Pertama, kita membuat variabel yang menampung alamat server basis data MySQL, username, password, dan nama basis data yang akan digunakan. Pada konfigurasi server MySQL saya, alamatnya adalah localhost, usernya root dan password-nya kopi.
$host = “localhost”;
$user = “root”;
$password = “kopi”;
$db = “petani”;
Fungsi mysql_connect() berfungsi menghubungkan PHP dengan MySQL.
$koneksi = mysql_connect($host, $user, $password);
Memeriksa, apakah sudah terhubung atau tidak.
if ($koneksi)
   echo “Sukses terhubung ke database MySQL”;
else
   die(“Gagal terhubung dengan database: <code style=‘color:red’>”.mysql_error().”</code>”);
Memilih basis data petani
mysql_select_db($db, $koneksi);
Menjalankan query dan menghitung rekord. Query adalah kalimat permintaan untuk mengakses data.
$kueri = mysql_query(“SELECT * FROM anggota”, $koneksi);
$rekord = mysql_num_rows($kueri);
Mencetak jumlah record
echo “Jumlah rokord: $rekord”;
Menutup koneksi
mysql_close($koneksi);

Uji Coba

Bukalah browser, kemudian arahkan ke alamat http://localhost/konek.php.

Sekarang, coba ganti password atau username pada kode php. Misalkan kopi diganti menjadi teh, maka koneksi akan gagal karena password salah.


Demikian panduan menghubungkan PHP dengan MySQL ini. Selanjutnya anda bisa mempelajari cara menampilkan isi tabel MySQL di PHP.
