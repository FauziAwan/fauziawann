Cara Membuat Desklet Linux Mint Sendiri

Desklet adalah pernak-pernik untuk menghias desktop Linux Mint, mirip gadget di Windows.

Kita bisa mencari dan men-download Desklet Linux Mint dari , akan tetapi Desklet yang saya cari tidak ada di sana.

Saya ingin Desklet untuk membuat hitungan mundur hari (countdown).

Misalnya seperti ini:

![Contoh CountDown days]()

Setelah pusing mencari di Google, akhirnya saya memutuskan untuk mencoba membuat Desklet sendiri.

Membuat Desklet Baru

Buat direktori baru:

mkdir -p ~/.local/share/cinnamon/desklets/deadline-remainder@petanikode.com

Buka dengan teks editor:

atom ~/.local/share/cinnamon/desklets/deadline-remainder@petanikode.com
