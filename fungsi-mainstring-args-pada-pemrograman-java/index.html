Fungsi main(String[] args) pada Pemrograman Java

Ardianta Pargo

Fungsi main() adalah fungsi utama dalam program java. Fungsi main() harus ada dalam setiap program. Tampa fungsi ini, program tidak dapat bekerja. Fungsi main() secara lengkap dan formal ditulis seperti berikut ini.

public static void main(String[] args){
    // kode program di sini 
}
public: Hak akses fungsi ini adalah publik. Semua kelas dapat mengakses fungsi ini.
static: fungsi ini diset static
void: fungsi ini tidak mengembalikan apa-apa.
String[] args: Variabel atau parameter yang menyimpan argumen

Fungsi main() memiliki sebuah parameter args untuk menampung argumen dari user. Pada lingkungan CLI Linux/Unix, program sering diberikan pilihan-pilihan atau argumen. Sebagai contoh: ls –all. Berikut ini contoh program java dengan argumen.
Kode programnya:
class Programku{
    public static void main(String[] argumen){
        System.out.println(“Argumen yang diberikan:“);

        for(int i = 0; i < argumen.length; i++){
            System.out.println(argumen[i]);
        }

    }
}
