Memahami kode C++

Saya mulai dari perintah #include, perintah ini berfungsi untuk memasukan sesuatu yang dibutuhkan di program. Sesuatu itu bisa fungsi, variabel, kelas, konstanta, dan sebagainya. Menggunakan perintah #include ibarat memasukan kopi, gula, dan air hangat ke cangkir.

Perintah namespace std; berfungsi untuk menentukan cakupan (scope) yang dipakai. Pada contoh di atas, cakupan yang dipakai adalah std. Perintah namespace ibarat memilih merek kopi yang akan diseduh.
using namespace kopiABC;
Fungsi main merupakan fungsi utama tempat menulis kode peintah atau ekspresi. Fungsi inti dari sebuah program. Tampa fungsi ini, program tidak bisa dijalankan.
int main(){
    // kode perintah 1;
    // kode perintah 2;
    // kode perintah 3;
    // …
    // kode perintah n;

    return 0;
}
Fungsi int main() harus mengembalikan nilai (return 0), karena jenis nilai kembaliannya sudah ditentukan, yaitu int (integer). Fungsi main juga dapat tidak mengembalikan nilai dengan menghilangkan nilai kembaliannya. 
main(){
    // kode perintah 1;
    // kode perintah 2;
    // kode perintah 3;
    // …
    // kode perintah n;
}
Setiap kode perintah diakhiri dengan tanda titik koma (;). Perintah cout berfungsi untuk mencetak teks ke layar. Penambahan endl agar kalimatnya dibuat satu baris. Bisa juga dubuat dengan menggunakan \n di dalam kalimatnya.
cout << “Apa kabar?\n”;

Perkenalkan dirimu

Setelah mengetahui struktur penulisan kode dalam bahasa C++ dan cara membuat program, sekarang silahkan perkenalkan diri Anda dengan kode.
#include <iostream>
using namespace std;

main(){
    cout << “Assalamualaikum” << endl;
    cout << “Apa kabar semua?\n”;
    cout << “Perkenalkan nama saya “;
    cout << “Ardianta” << endl;
    cout << “Bisa dipanggil ” << “iyan” << endl;
    cout << “Saya dari Lombok” << endl;
    cout << “Senang berkenalan dengan kalian” << endl;

}
