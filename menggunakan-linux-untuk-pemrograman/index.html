Belajar Pemrograman Apapun Menggunakan Linux

Ardianta Pargo · 27 Jun 2017

Kenapa Saya Menggunakan Linux untuk Pemrograman? [Ini 5 alasannya]

#Motivasi #Windows

Jika kalian bertanya, kenapa saya menggunakan Linux untuk pemrograman?

Saya jadi ingin menceritakan, bagaimana saya dari awal mulai tertarik dan menyukai Linux sampai saat ini.

Tapi sepertinya akan membutuhkan waktu yang panjang…

Mungkin lain kali saja, (insya’allah) di tulisan yang berbeda.

Balik lagi ke pertanyaan, kenapa saya menggunakan Linux untuk pemrograman?

Ada beberapa alasannya:

1. Saya Tidak Suka Virus

Hal yang paling saya tidak suka saat menggunakan Windows adalah Virus. Setiap terkena, wajib install ulang dan konfigurasi ulang semuanya dari awal.

Saya tidak suka menggunakan Anti Virus, karena memori laptop saya tidak cukup banyak dan akan membuatnya terasa berat.

Sementara di Linux, saya merasa sangat nyaman dan terbebas dari Virus.

Hingga saat ini, laptop saya tidak pernah virusan lagi. Bahkan banyak teman-teman yang minta dihapusin virus dari laptop saya. Mungkin mereka kira Linux kebal dari segala virus.

Padahal itu salah.

Linux juga bisa terkena virus, bahkan belakangan ini berdar kabar kalau Virus Ransomware sudah menyerang beberapa server hosting yang menggunakan Linux (versi lama).

Sumber: blog.trendmicro.com

Meskipun begitu, sampai saat ini komputer saya masih aman dan tentram. Karena saya tidak menggunakan Wine, Kernel selalu ter-update, dan tidak menginstal sembarang software. Apalagi software closed source yang tidak jelas asal-uslunya.

2. Lingkungan Development di Sistem *Nix Lebih Bagus

Ini menurut pendapat saya pribadi, tapi pendapat ini sebenarnya sudah didukung sendiri oleh Microsoft dan Apple (cieee 😄).

Microsoft sepertinya mengakui, kalau lingkungan coding di sistem *nix (Linux dan Unix) itu lebih bagus. Buktinya, mereka menyertakan Ubuntu dalam Windows untuk keperluan Development (pemrograman).

Kabar baiknya, Microsoft juga sangat suka berkontribusi ke proyek open source.

Sementara untuk Mac, ya Mac OS bahan dasarnya dari Unix. Mac juga enak dipakai ngoding dan UI-nya bagus. Tapi sayangnya saya tidak punya cukup banyak uang untuk membelinya. Akhirnya tetap bertahan menggunakan Linux.

Kalaupun pakai Mac, nanti juga ada Linux-nya. Soalnya pakai Docker dan Vagrant untuk pengembangan web.

Teknologi Docker dan Vagrant menggunakan Linux, jadi kita dituntut harus paham sistem Linux.

3. Mengurangi Main Game dan Perbanyak Belajar Coding

Saya memang seorang pecandu Game. Waktu luang banyak dihabiskan untuk bermain Game. Bahkan dulu rela berjalan berkilo-kilo untuk pergi main PS (karena waktu itu belum banyak yang punya PS)

Prinsip saya, “Kalau belum tamat, belum mau berhenti main”.

Prinsip tersebut yang membuat saya terus-terusan bermain Game.

Lalu, setelah punya rasa ketertarikan untuk belajar komputer, minat saya terhadap Game mulai berkurang…

Di kampus tempat saya kuliah, banyak sekali Gamer berkeliaran. Dulu pernah ikut-ikutan bermain game online saat masih menggunakan dual boot (Windows dan Ubuntu).

Tapi semuanya berubah, setelah 100% pakai Linux…

Game tidak lagi bisa diinstal di Laptop, sekarang fokusnya oprek Linux dan belajar Coding.

Stop Playing, Start Creating…

Bahkan sekarang punya ketertarikan untuk belajar Game Programming. Sedikit demi sedikit, saya mulai belajar pemrograman grafis dengan OpenGL, SDL, HTML5 Canvas, dan juga Game Engine.

Ini screenshot, saya lagi belajar Godot:

Kabar baiknya, sekarang Game Engine Unity sudah mulai didukung di Linux. Meskipun sudah ada Godot Engine yang open source dan siap bersaing dengannya.

Kita tunggu saja perkembangannya…

Saat ini saya kebanyakan menggunakan Linux untuk pemrograman Web dan desain.

4. Masih Penasaran dengan Linux

Saya sering punya rasa penasaran yang tinggi terhadap sesuatu yang baru dan belum pernah saya ketahui. Karena itu, saya menggunakan Linux.

Waktu saya pakai Windows, Saya menganggap Linux adalah sesuatu yang sulit dipelajari. Tapi ternyata tidak.

Walaupun pada awalnya banyak mendapatkan error dan masalah, saya masih bisa bertahan hingga saat ini.

Meskipun sadah lama menggunakan Linux, rasa pensaran saya masih ada. Karena masih banyak yang belum diketahui.

Seperti:

  • Bagaimana kernel bekerja?
  • Bagaimana cara compile kernel?
  • Bagaimana cara bikin distro linux sendiri?
  • Bagaimana Google mengembangkan Kernel Linux menjadi Android?

Sampai saat ini, saya belum paham, karena belum mencobanya.

5. Berhenti Membajak

Pembajakan software mungkin saudah menjadi hal yang biasa di negara kita. Mau install ini, mau install itu… tinggal cari saja crack-nya.

Tapi kalau terus-terusan melakukan ini, kapan mau maju?

Selama masih ada yang membuang sampah ke sungai dan merusak alam, bajir akan terus terjadi.

Kira-kira seperti itulah perumpamaannya.

Saya tidak punya kuasa untuk menghentikan mereka yang membuang samapah di sungai dan merusak alam. Karena itu, saya memilih untuk tidak ikutan lagi membuang sampah dan merusak alam.

Menggunakan FOSS tidak hanya gratis, melainkan untuk kebebasan dari perbudakan. . . #linux #foss #opensource #gnu

Akhir Kata…

Untuk saat ini, itulah 5 alasan saya menggunakan Linux untuk pemrograman.

Jika kalian menggangap saya “Linux Fans Boy”, “Maniak Linux”, “Fanatik Linux”, dan semacamnya. Semua itu tidaklah benar, karena saya punya rasa ketertarikan terhadap banyak hal termasuk juga Windows. Buktinya blog ini tidak bisa fokus membahas satu topik.

Bisa jadi saat ini saya masih pakai Linux, mungkin 5 atau 10 tahun mendatang saya sudah pakai Mac OS (ngimpi 😄).

Apa kamu juga tertarik menggunakan Linux atau sudah menggunakan Linux?

Mari berbagi di komentar…
