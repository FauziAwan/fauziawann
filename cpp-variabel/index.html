Belajar Pemrograman C++: Variabel dan Tipe Data

#cpp


Apa itu Variabel dan Tipe Data?

Variabel merupakan sebuah nama lokasi penyimpanan di memori. Variabel berperan sebagai wadah, tempat menyimpan data sementara. Data yang tersimpan di dalam variabel memiliki jenis yang beragam. Jenis-jenis data itu, disebut tipe data. Pada umumnya tipe data di permograman C++, dikelompokkan menjadi lima macam, tipe data teks, angka atau bilangan, boolean, pointer, dan void. Tipe data teks terdiri dari string dan char. Sementara tipe data angka terdiri dari int (integer), float, double, dsb.

Program Variabel

Setelah memahami apa itu variabel dan tipe data, mari kita mencoba membuat program dengan memanfaatkan variabel. Variabel dalam pemrograman C++ dapat dibuat di dalam fungsi main() atau pun di luar. Penulisan atau pembuatan variabel diawali dengan menulis tipe data terlebih dahulu, kemudian diikuti nama variabelnya

1. Kode Program

Buatlah sebuah file baru dengan nama variabel.cpp, kemudian isi dengan kode program berikut ini:
#include <iostream>

using namespace std;

int main(){
 
  // deklarasi tipe data variabel
  string nama;
  int umur;
  char jenisKelamin;
 
  // —— proses input ——
  cout << “Siapakah namamu?” << endl;
  cout << “jawab: “;
  // menyimpan data ke variabel
  getline(cin,nama);
 
  cout << “Berapa umurmu?” << endl;
  cout << “jawab: “;
  // menyimpan data ke variabel
  cin >> umur;
 
  cout << “Jenis kelamin [L/P]: “;
  // menyimpan data ke variabel
  cin >> jenisKelamin;
 
  // —— proses output ——
  cout << “Salam kenal, ” << nama << “ Sekarang engkau berusia “;
  cout << umur << “ dan kau berjenis kelamin “<< jenisKelamin;
 
  return 0;

}

2. Kompilasi

Kompilasi program dengan perintah ini:
g++ variabel.cpp -o variabel
Kemudian eksekusi dengan perintah ini:
./variabel

Kesimpulan

Variabel sangat berguna untuk menampung data sementara di memori. Jenis data yang tersimpan pada variabel disebut tipe data. Secara umum, tipe data pada C++ dikelompokkan menjadi lima yaitu: angka (integer, float), teks (char, string), boolean, void dan pointer.
