pemrograman vala

Apa itu Vala?

Vala adalah bahasa pemrograman berorientasi objek (OOP) dengan self-hosting compiler yang menghasilkan kode dalam bahasa C.

Secara tidak langsung, program yang ditulis dengan vala performanya sama dengan program yang ditulis dengan C — valadoc.org

Install Vala di Linux

Debian, Ubuntu, Linux Mint, ElementaryOS, dan keturunan lainnya:

sudo add-apt-repository ppa:vala-team
sudo apt-get update
sudo apt-get install valac

Archlinux:

sudo pacman -Sy
sudo pacman -S vala

Membuat Program Vala Pertama

touch hello_wolrd.vala

Jalankan program vala:

vala hello_wolrd.vala

Kompilasi program vala

valac hello_wolrd.vala
