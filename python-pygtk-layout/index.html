Pemrograman PyGTK: Mengatur Tata Letak dengan Kontainer

Ardianta Pargo

Kontainer merupakan komponen penting kedua yang diperlukan setelah jendela. Karena kontainer dapat kita gunakan untuk mengatur tata letak komponen pada jendela.

Ada beberapa jenis kontainer yang disediakan PyGTK:

  1. Fixed
  2. Table
  3. Alignment
  4. VBox
  5. Layout

Mari kita bahas satu-per-satu…

1. Kontainer Fixed

Kontainer fixed merupakan kontainer yang tata letaknya sudah ditentukan atau tidak dinamis alias kaku.

Kontainer fixed dibuat dengan membuat objek dari kelas Fixed().

Contoh:

kotainer_fixed = gtk.Fixed()

5. Kontainer Layout

Kontainer layout mirip seperti kontainer fixed, tapi kontainer layout memiliki area gulungan (scroll area) yang tak terbatas (lebih kecil dari 232).

Pada sistem X Window, ukran lebar dan tinggi jendela maksimal dibolehkan sampai 32767 piksel. Kontainer layout dapat mencapai sekitar batasan ini.

Referensi: http://www.pygtk.org/pygtk2tutorial/sec-Layout.html
