Pemrograman Android: Membuat Tombol FAB

Ardianta Pargo

FAB (Floating Action Button) merupakan sebuah tombol melayang yang biasanya terletak di kanan bawah layar. Tombol FAB mulai dikenal sejak peluncuran desain material (Material Design) oleh Google. Tombol FAB biasanya digunakan untuk aksi tertentu, seperti membuat pesan baru, menambah item tertentu, dsb.

1. Buat Proyek Baru

Buatlah sebuah proyek baru dengan Blank Activity di Anrdoid Studio.

2. Tambah Depedency

Buka file build.gradle kemudian tambahkan depedency desain material.
compile ‘com.android.support:design:22.2.0’
