Pengenalan Slim Framework untuk Pemula

#Slim #PHP #Framework #API

Slim merupakan sebuah framework mikro untuk PHP. Slim bisa kita gunakan untuk membangun aplikasi web sederhana dan Rest API. Kali ini, saya akan membawakan tutorial cara menginstal Slim di Linux. Slim yang akan diinstal adalah versi 2.0.

Pastikan Composer dan PHP sudah terinstal di komputer teman-teman. Setelah itu ikuti langkah-langkah berikut ini.

Membuat Direktori Proyek

Buatlah sebuah direktori dengan nama proyeknya, misalnya proyek-slim.

mkdir proyek-slim
cd proyek-slim

Instalasi Slim dengan Composer

Ketik perintah berikut untuk meinginstal Slim.

composer require slim/slim:~2.0

Perintah ini akan mendownload Slim dan menyimpannya di dalam direktori vendor.

Membuat File index.php

Silahkan buat file index.php di dalam direktori proyeknya dengan isi sebagai berikut.

<?php

require 'vendor/autoload.php';

$app = new \Slim\Slim();

$app->get("/salam/:nama", function($nama){
    echo "Assalamualaikum, $nama";
});

$app->run();

Sekarang kita punya struktur proyek seperti ini.

proyek-slim/
├── composer.json
├── composer.lock
├── index.php
└── vendor
...
14 directories, 62 files

Bagus, selanjutnya kita bisa menggunakan server bawaan PHP.

Menjalankan Server PHP

Ketik perintah berikut untuk menjalankan server PHP pada localhost dengan nomer port 8000.

php -S localhost:8000

Setelah itu, cobalah buka web browser dan arahkan ke alamat tersebut. Kalau hasilnya not found berarti rute / belum kita buat. Cobalah untuk mengakses rute /salam/.

Kita sudah berhasil menginstal Slim, selanjutnya kita bisa lakukan pengembangan lebih lanjut. Sekian dulu untuk kali ini.
