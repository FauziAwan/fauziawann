Emulator alternatif untuk Android Studio

Ardianta Pargo dalam pemrograman
13 November 2015 · 21:05

Saat mencoba Android Studio di Xubuntu 15.04 32bit, emulatornya tidak bisa bekerja dengan baik. Mungkin karena arsitekur CPU-nya tidak mendukung.

Pada pesan error yang muncul, saya dianjurkan menjalankan emulator dengan argumen -force-32bit. Tapi setelah itu, keluar lagi pesan error “KVM is not installed on this machine”.

Teman saya menyarankan menggunakan Genymotion sebagai emulator.

Allhamdulillah, sekrang sudah bisa. Berikut ini adalah beberapa hal yang saya lakukan.

1. Pastikan virtualbox sudah terinstal

Genymotion memanfaatkan virtualbox untuk melakuan virtualisasi. Jadi pastikan virtualbox sudah terpasang di komputer anda.

2. Download dan instal genymotion

Download Genymotion di website resminya (genymotion.com). Anda harus memiliki akun agar bisa mendownloadnya. File yang saya dapatkan adalah file bin (binary).

Setelah mendapatkan genymotion, selanjutnya menginstalnya ke direktori /opt. Saya pilih direktori ini, karena banyak aplikasi lain juga terinstal di sana. Sebenarnya bisa diinstal di mana saja, sesuka anda.

Gambar di atas adalah kumpulan file genymotion setelah terinstal di direktori /opt. Untuk menginstalnya di sana, kita hanya perlu mengeksekusi file genymotion-2.5.2_x86.bin di direktori /opt.

Sekarang coba eksekusi atau jalankan genymotion. Kemudian, tambahkan perangkat virtual (virtual device). Penambahan perangkat virtual memerlukan koneksi internet untuk login dan mendownload perangkat virtualnya. Login dengan akun yang sudah anda buat di genymotion.com.

Barikut ini tangkapan layar saat salah satu perangkat virtual (mesin virtual) dinyalakan.

2. Tambahkan plugin Genymotion di android studio

Klik menu file kemudian pilih Settings. Pada jendela yang terbuka, di bagian sebelah kiri, pilih plugins. Cari plugin “genymotion”, bila tidak ditemukan klik browse. Kemudian, cari lagi plugin “genymotion”. Tindakan ini memerlukan koneksi internet. Setelah ditemukan, silahkan diinstal.

Setelah instalasi selesai, silahkan buka ulang Android Studio-nya.

3. Uji coba

Setelah plugin genymotion berhasil terinstal, maka akan ada tambahan tombol Genymotion di toolbar Android Studio. Bila diklik, maka akan keluar jendela Genymotion Device Manager.

Silahkan nyalakan salah satu mesin virtual Genymotion. Kemudian run salah satu proyek app di android studio, maka anda akan dihadapkan dengan jendela pemilihan mesin virtual atau emulator.

Karena salah satu mesin genymotion sedang dinyalakan, makan akan tampil pilihannya di sana. Setelah dipilih, maka aplikasi pun berhasil dikirim ke mesin virtual genymotion dan dijalankan.

Android Studio sudah bisa bekerja dengan Genymotion.
